Movie API Documentation


This is the server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

5 Allow new users to register /users POST CREATE A JSON object holding data about the new user to add, structured like: { name: "Kim", favoriteMovies: [] } A JSON object holding data about the new user that was added, including an ID: { id: 1 name: "Kim", favoriteMovies: [] },
6 Allow users to update their user info (username) /users/:id PUT UPDATE None A JSON object holding data about the user that was updated: { id: 1 name: "Kimmy", favoriteMovies: [] },
7 Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:id/:movieTitle POST CREATE None A text message indicating whether the movie was added to the user's favorite list
8 Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/:id/:movieTitle DELETE DELETE None A text message indicating whether the movie was removed from the user's favorite list
9 Allow existing users to deregister (showing only a text that a user email has been removed) /users/:id DELETE DELETE None A text message indicating whether the user was removed from the database
